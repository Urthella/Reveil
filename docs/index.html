REVEIL

AI-powered adaptive habit transformation.

Reveil combines behavioral science with adaptive AI to help users build sustainable habits and navigate addiction recovery. Mobile-first, privacy-respecting, built as the COMP4901 graduation project at Konya Food and Agriculture University.

React Native · Expo NestJS · TypeORM FastAPI · OpenAI SQLite / Postgres Firebase Auth

What it does

📋 Habits with depth

Categories, weekly targets, pause-until auto-resume, freeze days that protect streaks, sort, search, 8 quick-start templates.

🤖 Adaptive AI feedback

Locale + tone + category-aware prompts. Rule-based fallback when offline. Every response passes a safety filter.

📊 Real-time analytics

30-day consistency, streak math that survives freeze days, 90-day heatmap, per-category rollup, weekly digest.

🎮 Gamification

XP for completions and streaks, six streak badges with unlock toasts, level-up celebration, progress rings.

🔔 Smart reminders

Per-user quiet hours, weekly digest cron with email + push, deep-link to the right habit on tap.

🔒 Privacy first

JSON + CSV export & import, account deletion, anonymized AI payload. All sensitive endpoints rate-limited.

Architecture

Three-tier separation between a React Native client, a NestJS API, and a FastAPI AI engine. SQLite for one-command demos, Postgres + TypeORM migrations for production. See the architecture diagrams and the SRS-to-code matrix.

Try the demo

git clone <repo> reveil
cd reveil/backend && npm install && npm run seed && npm run start:dev
cd ../ai_engine && python -m pip install -r requirements.txt && uvicorn main:app
cd ../mobile && npm install && npm start

Tap “Continue as demo guest” on the login screen for a pre-seeded data experience — no Firebase project needed.
